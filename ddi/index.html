AGCO South America APIs

Dealer Data Integration

Introdução

O escopo do processo visa a integração de notas fiscais de venda, devolução de venda e transferência de operações que ocorram das revendas para o cliente final, transmitindo os dados para a AGCO. O processo padrão será definido com a intergração via API, porém, a solução de contingência consiste no envio manual de uma solução Web. A API DDI oferece programaticamente uma forma de envio para a AGCO das informações de venda por parte das concessionárias. Dessa forma, o sistema da concessionária pode enviar automaticamente as informações sem intervenção de um operador.
